# SQL Student Management System

## Project Overview

This project is a beginner SQL database project built using MySQL.

The system manages:
- Students
- Courses
- Marks

It demonstrates:
- Database creation
- Table creation
- Data insertion
- Filtering
- Sorting
- Aggregate functions
- JOIN operations
- GROUP BY analysis
